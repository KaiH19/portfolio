Projects

Forecasting La Liga Winners

Developed a system to predict La Liga winners using a decade of historical match and player data. Engineered season-level features like Elo ratings, squad value, and win rates. Trained MLP and Random Forest models to forecast top teams and simulate league tables. Deployed a DASH web app via Render to visualize predictions and allow season simulations. Evaluated model accuracy against past seasons and improved interpretability through data visualizations.

Fleet Management Dashboard

Built a full-stack dashboard for real-time GPS tracking and vehicle maintenance alerts. Integrated MapLibre GL JS and simulated telemetry data for live tracking. Used WebSockets for real-time updates and MongoDB to store vehicle history. Implemented rule-based alerts based on mileage and service intervals.

Online Auction System

Developed a real-time auction platform with live bidding using ASP.NET Core and SignalR. Enabled instant updates, dynamic closing, and bid tracking via Redis and PostgreSQL. Included secure user authentication, admin tools, and simulated payment integration.
